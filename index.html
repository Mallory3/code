Task 1: Create Headings

Headings are used to define sections in a webpage. There are six levels of headings, from h1 (largest) to h6 (smallest).

Task: Create headings of different levels below:

  • Use <h1> to <h6> to create headings of different sizes.
  • Make sure each heading is in its own <h1>, <h2>, etc.
  • Try varying the size of the text in your headings.

Task 2: Paragraphs and Text Formatting

Paragraphs are used to break text into sections.

Task: Write a paragraph of text:

  • Apply <strong> to a word to make it bold.
  • Apply <em> to a word to make it italicized.
  • Apply <u> to a word to underline it.

Task 3: Links

Links are used to navigate between different pages.

Task: Add a link to your favorite website:

  • Use the <a> tag to create a hyperlink.
  • Include the href attribute to specify the link's URL.
  • Use target="_blank" to open the link in a new tab, and add the rel="noopener noreferrer" attribute for security.

Task 4: Create a Button

The <button> tag is used to create clickable buttons on a webpage. You can use it to trigger actions such as submitting forms, opening links, or executing JavaScript functions.

Task: Create a button and apply some styles:

  • Use the <button> tag to create a button.
  • Style the button using CSS to adjust the background color, padding, and font size.
  • Optionally, add a type attribute (e.g., type="button", type="submit") to specify its behavior.

Task 5: Images

Images are added with the img tag.

Task: Add an open source image:

  • Use the <img> tag to insert images.
  • Use the src attribute to specify the image source URL.
  • Provide an alt attribute to describe the image for accessibility.

Task 6: Lists

There are two types of lists in HTML: ordered and unordered. Use the ul tag for unordered lists and the ol tag for ordered lists.

Task: Create both an unordered and ordered list below:

  • Use <ul> for unordered lists (bulleted lists).
  • Use <ol> for ordered lists (numbered lists).
  • Include list items inside <li> tags.

Task 7: <div> and <span>

The <div> and <span> tags are commonly used as containers to group content and apply styles. The key difference is that <div> is a block-level element, while <span> is an inline element.

Task: Practice using the <div> and <span> tags:

  • Use <div> to group content that should be displayed in a block, such as a section or container.
  • Use <span> to group content that should be displayed inline, such as part of a paragraph.
  • Apply different styles to each using CSS, such as background colors, padding, or font sizes.
